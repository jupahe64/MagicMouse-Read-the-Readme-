<h4>MagicMouse
</h4>
<h6>English</h6>
This application, once open, will give your mouse cursor a frictionless feel
+ the mouse will bounce of the screen border.
<b>Use at your own risk. This program won't end until you cosed it manually, 
which is a bit hard without a working cursor.
In the worst case you can press win+tab select the magic mouse window, with the arrow keys, 
hit enter and then alt+F4, this should end the nightmare.
I'm NOT responsible for any wrong mouse click you may do.</b>

This software has only been tested on win7 and win10
